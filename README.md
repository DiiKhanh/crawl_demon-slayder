## Endpoints:

### Get All Characters

```GET https://demon-slayder-api-crawl.onrender.com/api/v1``` 

> Query limit: ```GET https://demon-slayder-api-crawl.onrender.com/api/v1?limit=5``` 

| Keys  | Type |
| ------------- | ------------- |
| name  | String  |
| url  | String  |
| image | String |
